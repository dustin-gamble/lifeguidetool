# D&D Image Prompt Pack (Mossbrook Starter Adventure)

Use this file to generate polished art for your D&D pages.

## How to use
1. Generate each prompt below in your preferred image tool.
2. Save output files into `/Users/dustingamble/Documents/New project/lifeguidetool/dnd-assets/`.
3. Use the exact filenames listed so pages auto-load your generated images.
4. If a generated file is missing, the site falls back to the built-in SVG placeholder art.

## Style lock (paste into every prompt)
"Storybook fantasy illustration, family friendly, Dungeons-and-Dragons-inspired but original, cinematic lighting, expressive faces, readable composition, soft painterly detail, adventurous and hopeful tone, no gore, no horror, no text overlays, 16:9 composition unless portrait requested."

## Negative prompt (optional)
"photorealistic, gore, blood, scary monster horror, watermark, logo, text caption, blurry face, extra limbs, malformed hands"
